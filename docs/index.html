Quickstart

Get started in under a minute.

# Clone and install
git clone https://github.com/rmax-ai/harness-fit.git
cd harness-fit

# Install dependencies (requires Bun)
bun install

# Run tests
bun test packages/ apps/ benchmarks/repositories/

# Type check
bun run typecheck

API Keys

Set OPENAI_API_KEY, ANTHROPIC_API_KEY, and GOOGLE_API_KEY in your environment. Copy .env.example to get started.
